# refrralAccredian

## Environment Variables
Add the following variables in your `.env` file:

```env
DATABASE_URL=""
PORT=<PORT>
CLIENT_ID=<CLIENT_ID>
CLIENT_SECRET=<CLIENT_SECRET>
REDIRECT_URI=<REDIRECT_URI>
REFRESH_TOKEN=<REFRESH_TOKEN>
GMAIL_USER=<MAIL>
NODE_ENV=<>
MONGO_URI=<MONGODB_URI>
```
